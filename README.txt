1. Be sure you have installed python3. 
If you’re not sure, you can check it or install Python3 using this guide(https://goo.gl/rCjaGk) (we don’t need a virtual environments).
2. Install pip:	
	sudo easy_install pip
3. Install nltk library:	
	sudo pip3 install -U nltk
4. Download(https://goo.gl/gmkwYr) my repository(https://goo.gl/RWCYdm) and unzip it whatever you want.
5. In terminal:
	cd <directory where you unziped an archive>
	cd search_engine-master
	cd src
	python3 main.py
